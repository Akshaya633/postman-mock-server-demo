# Postman Mock Server Demo 

A project demonstrating API mocking and testing using Postman's Mock Server feature.

## What This Does
- Created a mock server environment in Postman
- Defined mock API endpoints with example responses
- Tested API behaviour without a live backend

## Concepts Covered
- Mock server setup and configuration
- Environment variables in Postman
- API testing workflows
- Simulating real API responses for development

## Why Mock Servers Matter
Mock servers allow developers to build and test API integrations
before the backend is ready — a key skill in real-world development.
